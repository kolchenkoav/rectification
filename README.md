# Ректификация спирта

Веб-приложение для расчёта параметров ректификации спирта.

## Возможности

- Расчёт абсолютного спирта из спирта-сырца
- Расчёт головных фракций и голов
- Расчёт товарного спирта
- Расчёт хвостов
- Отображение результатов в HTML-шаблоне

## Технологии

- Java 21
- Spring Boot 3.1.5
- Thymeleaf
- Maven

## Сборка и запуск

```bash
# Сборка проекта
mvn clean package

# Запуск приложения
mvn spring-boot:run
```

Приложение будет доступно по адресу: http://localhost:8080

## Тестирование

```bash
# Запуск всех тестов
mvn test

# Запуск тестов с отчётом
mvn test -DskipTests=false
```

Тесты находятся в директории `src/test/java/`

### Юнит-тесты сервиса

- `RectificationServiceImplTest` — тесты логики расчёта

### Интеграционные тесты контроллера

- `RectificationControllerTest` — тесты HTTP-эндпоинтов

## Использование

1. Откройте http://localhost:8080
2. Введите параметры:
   - Количество спирта-сырца (л)
   - Крепость спирта (%)
   - Мощность (кВт)
   - Вода в узле отбора (мл)
3. Нажмите "Рассчитать"
4. Просмотрите результаты
